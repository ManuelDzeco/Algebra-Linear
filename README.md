# Project Documentation for Algebra Linear

## Project Structure
- **src/**: Contains the source code for the project.
- **notebooks/**: Includes Jupyter notebooks for data analysis and experimentation.
- **tests/**: Unit tests for the project.
- **docs/**: Documentation files.

## Notebooks List
1. **Notebook1.ipynb**: Description of what this notebook does.
2. **Analysis.ipynb**: Analysis of the results, figures, and an explanation of methods used.

## Technologies
- **Python 3.x**
- **NumPy**
- **Pandas**
- **Matplotlib**
- **scikit-learn**

## Dependencies
To install the necessary dependencies, run:
```bash
pip install -r requirements.txt
```

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ManuelDzeco/Algebra-Linear.git
   cd Algebra-Linear
   ```
2. Install dependencies.
3. Open the Jupyter notebooks for analysis. Start Jupyter with:
   ```bash
   jupyter notebook
   ```
